# lab15_Khemrayev_Yalkup

This repository contains the React + Vite application for lab work 15.

## Overview

- Built with Vite and React
- Includes ESLint and standard Vite development tooling
- Based on the React + Vite template

## Project structure

- `src/`: application source code
- `public/`: static assets
- `package.json`: project scripts and dependencies
- `vite.config.js`: Vite configuration

## Running the app

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the displayed local URL in your browser.
